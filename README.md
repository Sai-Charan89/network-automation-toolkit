# Network Automation & Troubleshooting Toolkit

This project provides Python scripts to help network engineers automate troubleshooting tasks in enterprise environments.

## Features
- DNS validation tool
- TCP connectivity checker
- Palo Alto firewall log parser

## Use Cases
- Troubleshooting network connectivity issues
- Validating DNS resolution
- Checking TCP sessions
- Analyzing firewall logs

## Technologies
- Python
- TCP/IP
- DNS
- Network Troubleshooting
